# Bee Pollen Management System 🐝🌿

Spring Boot backend system for managing plants and pollen data.

## 🚀 Tech Stack
- Java 17+
- Spring Boot
- Spring Security + JWT
- Spring Data JPA
- MySQL / PostgreSQL
- Docker

## 📦 Features
- Plant management (CRUD)
- Pollen management
- Authentication & Authorization (JWT)
- Image upload support
- Search by taxonomy (family, genus)

## 🛠️ Run project

```bash
mvn spring-boot:run
